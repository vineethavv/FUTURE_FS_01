# Personal Portfolio Website

## Project Overview
This is a responsive and modern Personal Portfolio Website developed using HTML, CSS, and JavaScript. The portfolio showcases my skills, education, projects, certificates, and contact information in a professional way.

The website is fully responsive and designed to provide a clean user experience on desktop and mobile devices.

---

## Features

- Responsive modern UI design
- Interactive navigation bar
- About Me section
- Education details
- Skills section
- Projects showcase
- Certificates section with preview option
- Contact form
- LinkedIn profile integration
- Download Resume button
- Smooth scrolling effects

---

## Technologies Used

- HTML5
- CSS3
- JavaScript
- GitHub Pages (Deployment)
